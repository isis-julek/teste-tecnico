# Carrossel de Benefícios - Teste Técnico

Este projeto consiste em um componente modular de "Carrossel de Benefícios". O objetivo principal foi atingir a fidelidade visual e implementar interações dinâmicas fluidas.

## 🚀 Tecnologias Utilizadas

* **HTML5 & CSS3:** Estrutura semântica e estilização.
* **Vanilla JavaScript:** Lógica de navegação e manipulação de estados sem dependências externas.
* **LottieFiles:** Integração de animações baseadas em vetores para os mascotes dos produtos.

## ✨ Funcionalidades e Diferenciais

* **Navegação Dinâmica:**
  * **Desktop:** Efeito de "Portal", onde o produto atual cai e o novo surge do topo.
  * **Mobile:** Transição de *swipe* lateral otimizada para o toque.
* **Responsividade Total:** Layout adaptável que transita de uma visualização horizontal complexa no desktop para uma organização vertical clara no mobile.
* **Estilização por Temas:** O fundo, as cores dos cartões e as sombras projetadas mudam dinamicamente de acordo com o produto selecionado (Vitamina D, Ferro ou Ômega 3).
* **Acessibilidade:** Implementação de `aria-labels` nos botões de navegação e uso de um `anunciador-acessibilidade` (`aria-live`) para informar usuários de tecnologias assistivas sobre a troca de produtos.
